# StellarRise HK Website (Next.js + Tailwind)

## Local dev
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Deploy (recommended: Vercel)
- Import the repo/folder into Vercel
- Framework: Next.js
- Build command: npm run build
- Output: default
- Add domain: stellarrisehk.com
